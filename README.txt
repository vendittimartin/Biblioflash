------ IMPORTANTE LEER ------

Al ejecutar el programa por primera vez se generará una carpeta llamada "Log" en la ubicación "...\Biblioflash\BiblioFlash UI\bin\Debug\net5.0-windows\Log".
Dentro de ella se generará a diario un archivo de texto (.txt) con nombre "log_año_mes_dia".
Estos archivos funcionarán como bitácora de monitoreo.
