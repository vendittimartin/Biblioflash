# Biblioflash 🚀

Development of a desktop application for the management of library loans in which the openlibrary api is consumed to register books.
It was developed under the application of good programming practices and under the use of design patterns.

Final project of the programming workshop subject.


------ IMPORTANTE LEER ------

Al ejecutar el programa por primera vez se generará una carpeta llamada "Log" en la ubicación "...\Biblioflash\BiblioFlash UI\bin\Debug\net5.0-windows\Log".
Dentro de ella se generará a diario un archivo de texto (.txt) con nombre "log_año_mes_dia".
Estos archivos funcionarán como bitácora de monitoreo.
